# 📊 Project Evaluator AI

Project Evaluator AI is a financial analysis tool that evaluates investment projects using key capital budgeting techniques.

## Features
- Payback Period
- Net Present Value (NPV)
- Internal Rate of Return (IRR)
- Modified IRR (MIRR)
- Profitability Index (PI)
- Project comparison (up to 5 projects)

## How to Use
1. Upload an Excel file with project cash flows
2. View computed financial metrics
3. Get the best project recommendation

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
